# Plan: Bass Guitar Neck WebGL Scene

## Goal
Draw a 3D bass guitar neck with 20 frets and 4 strings (E, A, D, G) using Three.js (via Scala.js facade `dev.cheleb:threesjs`). The neck should be draggable to rotate and scrollable to zoom using OrbitControls (already set up).

## Current State
- The project is a Scala.js app using Three.js via the `threesjs` facade (`dev.cheleb:threesjs:0.1.0`).
- `Main.scala` already has:
  - Scene, PerspectiveCamera, WebGLRenderer, lighting, OrbitControls (drag/rotate + scroll/zoom).
  - HUD elements (note display, title).
  - Animation loop.
- The scene is **empty** — no 3D objects are added yet.

## Implementation Plan

All changes will be in `src/main/scala/dev/cheleb/mythreeapp/Main.scala`. We'll add the neck geometry after the lighting setup and before the animation loop.

### Step 1: Create the Fretboard (Neck Wood)
- Use `BoxGeometry` for a flat rectangular board.
- Dimensions: width ~4 units (across strings), height ~0.3 (thin), depth ~20 units (along frets).
- Material: `MeshStandardMaterial` with a dark rosewood color (`0x3b1f0a` or similar).
- Position at origin, oriented so the neck extends along the Z-axis.

### Step 2: Create 20 Frets
- Use `BoxGeometry` (thin metal bars) for each fret.
- Dimensions: width slightly wider than the neck (~4.2), height ~0.05 (thin wire), depth ~0.05.
- Material: `MeshStandardMaterial` with metallic silver/nickel color (`0xcccccc`, high metalness).
- Position frets at intervals along the Z-axis. Use **equal spacing** (simplified) or realistic decreasing spacing based on the 12th-root-of-2 formula:
  - `fretPosition(n) = scaleLength * (1 - 1 / 2^(n/12))`
  - We'll use a scale length of ~20 units to fill the board.

### Step 3: Create 4 Strings (E, A, D, G)
- Use `CylinderGeometry` (very thin, tall cylinder laid on its side) for each string.
- Alternatively, use `BoxGeometry` with very small width/height for simplicity.
- Dimensions: radius ~0.02–0.04 (varying per string — thicker for lower strings), length = neck depth.
- Material: `MeshStandardMaterial` with steel color (`0xdddddd`, high metalness).
- Position 4 strings evenly spaced across the width of the neck, slightly above the fretboard surface.
- Strings: E (thickest), A, D, G (thinnest).

### Step 4: Add Fret Position Markers (Dots)
- Standard bass fret markers at frets: 3, 5, 7, 9, 12 (double), 15, 17, 19.
- Use `SphereGeometry` or `CylinderGeometry` (flat disc) for dot inlays.
- Material: `MeshStandardMaterial` with white/pearl color.
- Position on the fretboard face between the appropriate frets.

### Step 5: (Optional) Nut
- A bar at fret position 0 (the nut), slightly taller than frets.
- Material: bone/ivory color (`0xfffff0`).

### Step 6: Camera & Controls Tuning
- Adjust the camera initial position to give a good default view of the 20-fret neck.
- OrbitControls are already configured — drag rotates, scroll zooms. No changes needed unless we want to constrain rotation.

## Geometry Layout Summary

```
Z-axis (neck length, 0 → ~20)
├── Nut at Z=0
├── Fret 1 at Z = fretPos(1)
├── Fret 2 at Z = fretPos(2)
├── ...
└── Fret 20 at Z = fretPos(20)

X-axis (string spread, -1.5 → +1.5)
├── String E at X = -1.5
├── String A at X = -0.5
├── String D at X = +0.5
└── String G at X = +1.5
```

## Technical Notes
- All Three.js API calls use the Scala.js facade (`import THREE.*`).
- Available geometries confirmed: `BoxGeometry`, `CylinderGeometry`, `SphereGeometry`, `PlaneGeometry`.
- Available materials: `MeshStandardMaterial`, `MeshPhongMaterial`, `MeshBasicMaterial`.
- `Mesh(geometry, material)` creates a renderable object, added to scene via `scene.add(mesh)`.
- `Group()` can be used to group all neck components for easier manipulation.
