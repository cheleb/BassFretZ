# BassFretZ 🎸\n\nA 3D bass guitar fretboard visualizer built with [Scala.js](https://www.scala-js.org/), [Laminar](https://laminar.dev/) and [Three.js](https://threejs.org/).\n\n## Demo\n\n👉 **[Live demo on GitHub Pages](https://cheleb.github.io/BassFretZ/)**\n\n## Tech Stack\n\n- **Scala.js** — Scala compiled to JavaScript\n- **Laminar** — reactive UI framework for Scala.js\n- **Three.js** — 3D rendering in the browser\n- **Vite** — fast frontend build tool\n\n## Development\n\n```bash\n# Install JS dependencies\nbun install\n\n# Start dev server (with Scala.js watch)\nsbt ~fastLinkJS\nbun run dev\n```\n\n## Build\n\n```bash\nsbt fullLinkJS\nbun run build\n```\n
